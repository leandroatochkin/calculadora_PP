# Entrega Calculadora JS/CSS/HTML Prácticas Profesionalizantes


Una calculadora web diseñada para ser totalmente responsiva y funcional tanto con mouse como con teclado.

## Instalación y Uso

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/leandroatochkin/calculadora_PP
    ```
2.  **Abrir el proyecto:**
    Abrir el archivo `index.html` con tu navegador de preferencia.
